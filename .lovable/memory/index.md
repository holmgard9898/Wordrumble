# Project Memory

## Core
Word Rumble - bubble word puzzle game. Swedish UI text.
Game bg: linear-gradient dark purple/blue. Bubble colors: red, green, blue, yellow, pink.
ENABLE word list (no abbreviations/names). Lovable Cloud enabled.
No microtransactions - coins + ad-watching only. PWA installable.
Auth required for multiplayer (email + Google OAuth).

## Memories
- [Game modes](mem://features/game-modes) — Classic (50 moves), Word Surge (bonus moves), Bomb mode, 5+, Ett Ord
- [Multiplayer rules](mem://features/multiplayer) — Turn-based, 2-3 rounds per mode, 48h timeout, shared/unshared words, chat, max 20 matches

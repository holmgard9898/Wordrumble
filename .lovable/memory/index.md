# Project Memory

## Core
Word Rumble - bubble word puzzle game. Swedish UI text.
Game bg: linear-gradient dark purple/blue. Bubble colors: red, green, blue, yellow, pink.
ENABLE word list (no abbreviations/names). Lovable Cloud enabled.
No microtransactions - coins + ad-watching only. PWA installable.
Multi-language support: EN, SV, DE. Language-specific letter values & pools.

## Memories
- [Game modes](mem://features/game-modes) — Classic (50 moves), Word Surge (bonus moves), 5+ Bokstäver (100 moves, 3 colors), Ett Ord (60 moves, best word only), Bomb mode (word defuses bombs)
- [Multiplayer rules](mem://features/multiplayer) — 2 rounds, 25+50+25 moves, shared used words per round
- [Menu structure](mem://features/menu-structure) — Main menu, play, challenge, shop, statistics, settings
- [Planned features](mem://features/planned) — Äventyr (candy crush-style levels), friends, online multiplayer, shop items

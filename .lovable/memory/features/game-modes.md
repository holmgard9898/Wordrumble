## Classic
- 50 moves, standard 5 colors
- Length bonus: 4→+2, 5→+4, 6→+6, 7→+8, 8→+10, 9→2x, 10→3x letter points

## Word Surge
- 50 moves, standard 5 colors, no length bonus points
- Score bonus moves: ≥10pts → +10 moves, ≥15pts → +25 moves
- Length bonus moves: ≥5 letters → +10, ≥7 → +25, =10 → +50 (additive with score bonus)

## 5+ Bokstäver
- 100 moves, only 3 colors (red/green/blue), min 5-letter words
- Same length bonus scoring as Classic

## Bomb Mode
- No move limit, standard 5 colors
- Random vowels get bomb timers (min 10 moves), timer decrements each move
- 1-3 bombs from start, max 5 simultaneous, 30% chance to spawn new after word found
- Game ends when any bomb reaches 0
- No length bonus points (like Surge)
